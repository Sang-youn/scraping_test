하나씩 작업파일 및 작업내역 추가예정

# 세팅방법 
1. 환경변수 `CHROME_DRIVER_PATH`에 크롬드라이브 경로를 설정한다
- 예 : /Volumes/{PATH}/chromedriver-mac-arm64/chromedriver

2. 필요 패키지를 설치한다(`requirements.txt`에 기술되어 있다)
- pip3 install -r requirements.txt

3. 실행파일은 main.py 로 한다

# 참고사항
1. 본인의 크롬 버전과 크롬드라이브 버전이 맞아야 한다(2025.12.19일 기준 버전143)
